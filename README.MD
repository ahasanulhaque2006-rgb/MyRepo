#### CARD FLIPPING GAME 🧮

## DISCRIPTION
A fun memory-based card flipping game where players match pairs of cards.
Built using HTML, CSS, and JavaScript with smooth animations and responsive design.

## Features
- Flip card animation
- Match-pair logic
- Score tracking
- Different set of images/cards to match(9 sets)
- Responsive design

## INSTALLATION/HOW TO RUN 
 In bash , run :
 git clone https://github.com/ahasanulhaque2006-rgb/MyRepo.git
 and then run : 
 cd .\MyRepo\
 finally :
 open .\MemoryFlipper.html and have a good time !
 
 
## How to Play
1. Click on a card to flip it.
2. Flip another card to find its matching pair.
3. Match all pairs to win the game.
4. Complete the game in minimum moves/time.
